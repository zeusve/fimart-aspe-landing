## 🛠️ Tech Stack

```
Frontend     →  React 18 + TypeScript
Styling      →  Tailwind CSS + shadcn/ui
Animations   →  Framer Motion
Routing      →  React Router DOM
Build        →  Vite 5
Testing      →  Vitest
Deploy       →  Vercel
```

## 🚀 Quick Start

```bash
# Clonar
git clone https://github.com/zeusve/fimart-aspe-landing.git
cd fimart-aspe-landing

# Instalar
npm install

# Desarrollo
npm run dev
```
