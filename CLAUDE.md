# CLAUDE.md - Normas de Acceso del Proyecto

## Permisos Otorgados

Este proyecto tiene permisos completos para Claude Code:

### Bash / Terminal
- Ejecutar cualquier comando de terminal
- Instalar dependencias (npm, bun, yarn)
- Ejecutar scripts de desarrollo, build y test
- Operaciones de git (commit, push, pull, branch, merge)
- Crear, eliminar y modificar archivos/carpetas

### Archivos
- Leer cualquier archivo del proyecto
- Escribir y editar cualquier archivo
- Crear nuevos archivos y carpetas
- Eliminar archivos cuando sea necesario

### Git
- Crear commits
- Crear y cambiar ramas
- Push a repositorios remotos
- Crear Pull Requests

## Stack Técnico

- **Framework**: Vite + React
- **Lenguaje**: TypeScript
- **Estilos**: Tailwind CSS
- **Package Manager**: npm / bun

## Comandos Comunes

```bash
# Instalar dependencias
npm install

# Desarrollo
npm run dev

# Build
npm run build

# Lint
npm run lint
```

## Estructura del Proyecto

- `src/` - Código fuente
- `public/` - Assets estáticos
- `index.html` - Punto de entrada HTML
